3.2 Version control

  1. Git

    Some Important steps

    1. Before first commit to check the

    repo status 

    git status
    1. Add files to staging area
    2. To add a particular file in the staging area
    git add <file_name>
    • To add all files to the staging area
    git add -A
    1. Remove files from staging area
    git reset <file_name>
    1. To remove all files from the staging area
    2. To put all files in the staging area to the repository we have to perform a commit
    git commit -m "My first commit"
    1. To view logs
    git log
    1. Pull and push changes to remote repository
    git pull origin main

    git push origin main
